# Apache ZooKeeper (apache-zookeeper)
Apache ZooKeeper is a centralized service for maintaining configuration information, naming, providing distributed synchronization, and providing group services. It is used by many distributed systems for coordination and leader election.

## APIs

### Apache ZooKeeper
ZooKeeper provides Java and C client APIs for distributed coordination operations including znodes, watches, ephemeral nodes, leader election, and distributed locks, plus the Curator high-level Java API and a REST API via the admin server.

**Human URL:** [https://zookeeper.apache.org/doc/current/](https://zookeeper.apache.org/doc/current/)
